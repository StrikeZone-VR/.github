## 🚀 프로젝트 소개

**StrikeZone VR은 KBO 리그를 '보는' 것을 넘어, 팬들이 직접 경기를 '경험하는' 새로운 VR 콘텐츠입니다.** 

**Unity의 XR Interaction Toolkit을 통해 팬들은 실제와 흡사한 투구와 타격감을 생생하게 체험할 수 있으며  
웹 크롤링을 통해 실시간 경기 데이터를 VR환경에 시각화하고 이를 팬들이 함께 공유하며 소통할 수 있는 커뮤니티 환경을 제공합니다.**  

**이처럼 StrikeZone VR은 팬들의 리그에 대한 몰입도를 높이고 강력한 팬 문화를 형성하는 데 기여할 것입니다.**
